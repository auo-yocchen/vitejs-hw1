# 我的最愛小工具

## 使用方式
1. 輸入要存網址
2. 輸入要存的標籤名稱
3. 按覺得最像的那一顆鈕存起來
4. 按清空可以清空之前存過的所有東西
5. 按存過的網頁旁邊的刪除可以刪掉那一個
